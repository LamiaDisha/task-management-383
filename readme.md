# Task Management API (Node.js + Express)

A simple REST API for managing tasks using Express.js with in-memory data storage.  
Provides task retrieval, creation, health checking, and error handling.

---

##  Getting Started

### **Prerequisites**
- Node.js (v16 or higher recommended)
- npm (comes with Node)

### **Installation**
```bash
git clone <your-repo-url>
cd task-management
npm install
